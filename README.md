# Проект по определению возраста покупателей

##  Задача
Сетевой супермаркет «Хлеб-Соль» внедряет систему компьютерного зрения для обработки фотографий покупателей. Фотофиксация в прикассовой зоне поможет определять возраст клиентов, чтобы:

1. Анализировать покупки и предлагать товары, которые могут заинтересовать покупателей этой возрастной группы
2. Контролировать добросовестность кассиров при продаже алкоголя

Необходимо построить модель, которая по фотографии определит приблизительный возраст человека. В распоряжении набор фотографий людей с указанием возраста.
Метрика MAE должна быть меньше 8

## Датасет
Общедоступный датасет APPA-REAL, состоящий из 7591 изображений людей различного пола, расы, возраста и выражения лица, с двумя метками - реальный возраст и примерный возраст (составленный на основе голосов людей, которым показали фотографию)

## Достигнутый результат
MAE метрика для модели составила 5.9650
Эталоном из предоставленной заказчиком исследовательсокй работы является метрика в 5.4, при этом метрика ниже 7 определялась заказчиком как "отличный результат"

## Использованная модель
В проекте была использована ResNet50 по желанию заказчика
